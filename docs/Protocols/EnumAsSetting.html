EnumAsSetting

public protocol EnumAsSetting

Enums that implement EnumAsSettingProtocol can be used as values for EnumSetting.

The NoiseTagging framework provides extensions so that enums that use Int for raw values automatically conform to this protocol, except for var displayName.
