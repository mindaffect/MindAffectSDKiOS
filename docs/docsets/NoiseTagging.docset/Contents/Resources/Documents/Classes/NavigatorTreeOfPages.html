NavigatorTreeOfPages

public class NavigatorTreeOfPages : NSObject, NoiseTagDelegate, PageDelegate

This class makes it easy to implement UI similar to what UINavigationController provides, but controllable via brain. The content displayed in a UINavigationController is defined using pages, see Page.

Pages can be:

  • List pages, which have sub pages where the user can navigate to;
  • Settings pages, which provide automatically generated UI to edit settings (see Setting), or

  • View pages, which simply provide a UIView to be displayed.

Important

Currently NavigatorTreeOfPages assumes that the device orientation does not change. Furthermore, Settings pages assume the orientation is landscape.
