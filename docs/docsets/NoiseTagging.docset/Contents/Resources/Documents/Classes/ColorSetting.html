ColorSetting

public class ColorSetting : Setting_abstract, Setting

A Setting with a color value.

The color is defined by one of two options:

  1. It is one of the ‘pickable colors’, which are predefined colors (PickableColor) as set on var pickableColors.
  2. It is specified using RGBA.
